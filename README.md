# DisCraft
![GitHub](https://img.shields.io/github/license/boymelancholy/DisCraft?style=for-the-badge)
![GitHub release (latest by date)](https://img.shields.io/github/v/release/boymelancholy/DisCraft?style=for-the-badge)
  
Reverting item which crafted!

## Version on debugged
1.12.2

## Overview
You can revert items which crafted before.  
Don't mind about waste items anymore by this plugin!  

## How to use
 1. Hold the item you want revert in hand.
 2. Do sneaking.
 3. Tap the Workbench block with **LEFT** click.
 4. Look at your inventory.
